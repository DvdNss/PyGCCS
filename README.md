## Getting Started

### Installation

1. Clone the repo

```shell
git clone https://github.com/DvdNss/PyGCCS.git
```

2. Move to the repo folder
```shell
cd PyGCCS
```

4. Install requirements

```shell
pip install -r requirements.txt
```
